איך משתמשים?

1

הורד את הקובץ

לחץ על כפתור ההורדה למעלה כדי לקבל את קובץ הדוגמה

2

מלא את הפרטים

פתח את הקובץ באקסל ומלא את הקטגוריות וההוצאות הקבועות שלך

3

שמור את הקובץ

שמור את הקובץ במחשב או בטלפון שלך

4

שלח לבוט

שלח את הקובץ לבוט בוואטסאפ - הוא יייבא הכל אוטומטית

נקודות חשובות

  • אל תמחק את שורת הכותרות (שורה 1)
  • הקפד למלא את כל העמודות הנדרשות
  • שם הקטגוריה בהוצאות קבועות חייב להתאים לשם קטגוריה קיימת
  • ניתן להוסיף כמה שורות שתרצה
  • הקובץ חייב להישאר בפורמט .xlsx

שאלות נפוצות

מה קורה אם שם הקטגוריה כבר קיים?

הבוט יעדכן את התקציב של הקטגוריה הקיימת במקום ליצור אחת חדשה.

האם אני חייב למלא גם קטגוריות וגם הוצאות קבועות?

לא! אפשר למלא רק קטגוריות, רק הוצאות קבועות, או את שניהם. הבוט יייבא רק את מה שמלאת.

מה קורה אם יש טעות בקובץ?

הבוט יודיע לך על שגיאות ויסביר מה צריך לתקן. לא תאבד מידע - פשוט תקן ושלח שוב.

כמה פעמים אפשר להשתמש בייבוא?

ללא הגבלה! אפשר לייבא קבצים כמה פעמים שתרצה, בכל תדירות שנוחה לך.

מוכנים להתחיל?

הורד את הקובץ והתחל לנהל את ההוצאות שלך בקלות
